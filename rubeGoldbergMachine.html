Rube Goldberg Machine

Video of Rube Goldberg Machine

A Rube Goldberg Machine is a machine designed to perform a simple task as inefficiently as possible. The main goal for a machine like that is for it to be able to complete as many actions as possible before ultimately accomplishing the task that it’s meant to perform. For this machine, the main objective was to hit a gong as the final action (since we didn’t have a gong, we substituted it with a tambourine) and was made for the Western Science Olympics in 2014.

To create the Rube Goldberg Machine, various objects were incorporated into the machine such as household toys and random materials from the dollar store. Some of these materials included golf balls, wooden blocks and paper cups. The video on the left shows what the machine looked like after it was completely built and the sequence of events that occurs before it hits the tambourine.
