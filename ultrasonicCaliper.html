Ultrasonic Caliper

Fully Assembled Ultrasonic Caliper

The “Ultrasonic Caliper” was created for a project for the course MTE 201 (Experimental Measurement and Statistical Analysis). The objective of the project was to create a device that is able to take a measurement, such as length or temperature, while also implementing a sensor into the device. Our project involved the use of an ultrasonic sensor to measure distance and an Arduino to provide power for the ultrasonic sensor and to interpret the data from the ultrasonic sensor. The above picture shows what the “Ultrasonic Caliper” looked like after it was completely built.


The caliper functions by adjusting the wooden block so that the object being measured is wedged between the wooden block and the acrylic box. The ultrasonic sensor then sends pulses to bounce off the object. The ultrasonic sensor is able to detect the pulses that have been bounced off the object and is able to determine how far away the wedged object is from the sensor (the sensor’s location is treated as the reference point). This information is then interpreted by the Arduino’s program to convert it into a metric and imperial measurement.


To create the acrylic box and the wooden block, the various parts were designed in AutoCAD and then created using a laser cutter. The caliper was then assembled using super glue and the electrical components of the caliper were encased within the acrylic box. Finally, the program for the Arduino was written in C++ and written within the native Arduino IDE.
